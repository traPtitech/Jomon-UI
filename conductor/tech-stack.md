# Technology Stack

## Frontend Framework

- **Core:** Vue 3 (Composition API)
- **Language:** TypeScript
- **Build Tool:** Vite 7

## State Management

- **Library:** Pinia (Setup Store pattern)

## Styling

- **Framework:** Tailwind CSS 4

## UI Component Libraries

- **Headless UI:** Used for complex accessible components like SearchSelect and SearchMultiSelect. (Replaced Reka UI for better type safety).

## Network Layer

- **HTTP Client:** axios
- **API Integration:** OpenAPI Generator (TypeScript client)

## Testing

- **Unit Testing:** Vitest
- **E2E Testing:** Playwright
